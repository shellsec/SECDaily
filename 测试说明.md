# AI总结功能测试说明

## 测试方法

### 方法1: 运行完整测试脚本（推荐）

运行测试脚本，会自动测试所有功能：

```bash
python test_ai_summary.py
```

测试脚本会测试：
1. ✅ oneapi.py 的 AI 总结功能
2. ✅ 保存AI总结到文件
3. ✅ HTML生成时加载AI总结
4. ✅ yarb.py 集成功能

### 方法2: 手动测试步骤

#### 步骤1: 测试 oneapi.py

```bash
python oneapi.py
```

或者直接测试总结功能：

```python
from oneapi import summarize_security_news

# 读取测试文件
with open('archive/2025/2025-11-06.md', 'r', encoding='utf-8') as f:
    content = f.read()

# 调用AI总结
result = summarize_security_news(content)
print(result)
```

#### 步骤2: 测试 yarb.py 集成

运行 yarb.py，它会自动：
1. 生成当天的 archive 文件
2. 调用 AI 总结
3. 保存到 AISummary+日期.md

```bash
python yarb.py --force
```

#### 步骤3: 测试 HTML 生成

运行 convert_today.py，它会：
1. 查找当天的 markdown 文件
2. 查找对应的 AI 总结文件
3. 生成包含 AI 总结的 HTML 文件

```bash
python convert_today.py
```

#### 步骤4: 查看页面效果

打开生成的 HTML 文件，检查右侧浮窗是否显示 AI 总结：

```bash
# 打开浏览器查看
start archive/2025/2025-11-06.html
```

## 常见问题

### Q: AI总结返回错误
A: 检查 `ai_config.py` 中的配置是否正确，确保API密钥和URL有效。

### Q: HTML文件没有显示AI总结
A: 确保：
1. AI总结文件已生成（AISummary+日期.md）
2. 文件在同一个目录下
3. 运行了 convert_today.py 重新生成HTML

### Q: 浮窗不显示
A: 检查：
1. 浏览器控制台是否有JavaScript错误
2. 模板文件是否正确更新
3. AI总结内容是否正确加载
